# 공급망 관리 화면 통합 완료

## 📋 개요

AIFIX 협력사 포털의 공급망 관리 화면을 모든 협력사 차수(1차/2차/3차)가 **동일한 UI**를 사용하도록 통합했습니다.  
이제 차수에 따라 화면이 달라지는 것이 아니라, **권한만 다르게 동작**합니다.

---

## 🎯 설계 목표

### 기존 구조 (Before)
- **1차 협력사**: "공급망 관리" 탭 (`Tier1SupplyChainManagement`)
- **2차/3차 협력사**: "공급망 조회" 탭 (`NonTier1SupplyChainView`)
- ❌ 차수별로 다른 화면과 컴포넌트 사용

### 새로운 구조 (After)
- **모든 차수**: "공급망 관리" 탭 (`SupplyChainManagement`)
- ✅ 동일한 UI, 권한만 다르게 동작
- ✅ SRM 시스템 스타일의 3-패널 레이아웃

---

## 🏗️ UI 구조

### 3-패널 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│  좌측 (3 cols)   │   중앙 (6 cols)   │   우측 (3 cols)   │
│                  │                   │                   │
│  공급망 트리      │   협력사 목록      │   협력사 상세      │
│  - 검색          │   - 필터 탭       │   - 기본 정보     │
│  - 펼치기/접기    │   - 테이블        │   - 제출 상태     │
│  - 상태 아이콘    │   - 클릭 선택     │   - 이력          │
│                  │                   │   - 액션 버튼     │
└─────────────────────────────────────────────────────────┘
```

## 🔐 권한 제어

| 협력사 차수 | 하위 협력사 초대 | 데이터 요청 | 데이터 직접 수정 |
|-----------|----------------|-----------|----------------|
| **1차**   | ✅ 가능         | ✅ 가능    | ✅ 가능         |
| **2차**   | ✅ 가능         | ✅ 가능    | ❌ 불가능       |
| **3차**   | ✅ 가능         | ❌ 불가능  | ❌ 불가능       |
| **4차+**  | ✅ 가능         | ❌ 불가능  | ❌ 불가능       |

### 주요 특징
- **모든 협력사**가 하위 협력사를 초대할 수 있습니다 (4차, 5차 등 제한 없음)
- **1차**만 하위 협력사 데이터를 직접 수정할 수 있습니다
- **2차**는 수정요청/보완요청만 가능합니다
- **3차 이상**은 조회 전용이지만, 하위 협력사 등록은 가능합니다

## 🎨 주요 기능

### 1️⃣ 좌측 패널 - 공급망 트리
- 계층 구조 시각화 (우리회사 → 2차 → 3차)
- 노드 클릭 시 중앙 협력사 목록 필터링
- 상태 아이콘 표시 (제출완료/미제출/보완필요/검증실패)
- 검색 기능
- 모두 펼치기/접기 버튼

### 2️⃣ 중앙 패널 - 협력사 목록
- 필터 탭 (전체/미제출/보완필요/검증실패)
- 테이블 컬럼:
  - 업체명
  - 차수
  - 상태
  - 최근 제출일
  - 마감일 (D-day)
- 클릭하여 우측 상세 패널에 정보 표시

### 3️⃣ 우측 패널 - 협력사 상세
- **업체 기본 정보**
  - 회사명, 차수, 상태, 제출일, 마감일
- **최근 이력**
  - 타임라인 형식으로 표시
- **주요 액션 버튼** (권한에 따라 표시)
  - 1차: "데이터 직접 수정" 버튼
  - 2차: "수정요청 보내기" 버튼
  - 공통: "데이터 보기", "보완요청 보내기", "재요청 보내기"

## 🧪 테스트 시나리오

### 1차 협력사
1. "공급망 관리" 탭 클릭
2. 좌측 트리에서 2차/3차 협력사 확인
3. 중앙 목록에서 협력사 선택
4. 우측 상세에서 "데이터 직접 수정" 버튼 확인
5. "하위 협력사 초대" 버튼 클릭하여 초대 모달 확인

### 2차 협력사
1. "공급망 관리" 탭 클릭
2. 좌측 트리에서 3차 협력사 확인
3. 중앙 목록에서 협력사 선택
4. 우측 상세에서 "수정요청 보내기" 버튼 확인 (직접 수정 불가)
5. "하위 협력사 초대" 버튼 클릭하여 초대 모달 확인

### 3차 협력사 (하위 협력사 없는 경우)
1. "공급망 관리" 탭 클릭
2. 중앙 패널에 "등록된 하위 협력사가 없습니다" 메시지 확인
3. "하위 협력사 초대" 버튼 **표시 확인** (모든 차수 공통)
4. 초대 버튼 클릭하여 4차 협력사 등록 가능

### 3차 협력사 (하위 협력사 있는 경우)
1. "공급망 관리" 탭 클릭
2. 좌측 트리에서 4차 협력사 확인
3. 중앙 목록에서 4차 협력사 선택
4. 우측 상세에서 조회만 가능 확인 (수정/요청 불가)

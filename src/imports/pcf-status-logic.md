AIFIX 협력사 포털 – PCF 상태 로직 수정

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 협력사 포털의 공급망 데이터 테이블에서 PCF 상태 표시가 실제 PCF 계산 로직과 맞지 않는다.

PCF 계산은 공급망 의존 구조(Upstream Dependency)를 가진다.

즉

하위 협력사의 PCF 계산이 완료되지 않으면
상위 기업은 PCF 계산을 수행할 수 없다.

현재 화면에서는 다음과 같은 불가능한 상태가 표시되고 있다.

예시

우리회사 → PCF 완료
하위 협력사 → 미제출

이 상태는 실제 시스템에서 발생할 수 없으므로
UI 상태 로직을 수정해야 한다.

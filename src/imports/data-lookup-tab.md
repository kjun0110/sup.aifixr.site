너는 제조업 공급망 ESG/PCF 관리 SaaS **AIFIX 협력사 포털**의 시니어 프로덕트 디자이너다.

현재 설계 대상은  
협력사 포털 → 프로젝트 선택 → 1차 역할 프로젝트 진입 → 네비게이션2 탭 중  
**“데이터 조회” 탭 화면**이다.

이 화면은 **1차 협력사용 화면이며 비1차 화면은 설계하지 않는다.**

--------------------------------
[고정 시스템 전제]
--------------------------------

AIFIX 협력사 포털은 다음 구조를 가진다.

- 단일 원청 전용 포털
- 협력사는 구매/PCF 직무를 분리하지 않고 **통합 템플릿 하나에 데이터 입력**
- 현재 사용자는 **1차 협력사**

1차 협력사 권한

- 하위차사 공급망 구조 조회 가능
- 하위차사 데이터 조회 가능
- 하위차사 데이터 직접 수정 가능 (단, 이 화면에서는 수정 금지)
- 전체 데이터 취합 후 원청 제출

--------------------------------
[업무 흐름 규칙]
--------------------------------

협력사 업무 흐름은 다음 구조를 따른다.

1️⃣ 데이터 관리  
→ 데이터 입력 및 수정

2️⃣ PCF 산정/전송  
→ PCF 계산 실행 및 전송

3️⃣ 데이터 조회  
→ 입력 데이터 및 계산 결과 확인

따라서 데이터 조회 탭은 **조회 전용 화면**이다.

이 화면에서는

- 데이터 수정
- 데이터 입력

기능을 제공하지 않는다.

--------------------------------
[화면 상단 공통 구조]
--------------------------------

네비게이션2 화면 상단에는 **프로젝트 컨텍스트 헤더**가 고정된다.

헤더 표시 정보

- 프로젝트명
- 내 차수 (1차)
- 제출 대상 (원청사)
- 계약번호
- 마감일
- 프로젝트 상태

그 아래에 네비게이션2 탭을 유지한다.

네비게이션2

대시보드  
공급망 관리  
데이터 관리  
데이터 조회 (현재 활성)  
PCF 산정/전송  
이력

--------------------------------
[데이터 조회 탭 설계 목적]
--------------------------------

데이터 조회 탭은 다음 목적을 가진다.

- 자사 입력 데이터 확인
- 하위차사 제출 데이터 확인
- PCF 계산 결과 확인
- 버전별 결과 비교
- 제출 전 최종 검토

중요 UX 규칙

입력 데이터와 계산 결과를 **같은 화면에서 섞어 보여주지 않는다.**

반드시 다음 두 영역으로 나눈다.

1️⃣ 입력 데이터 조회  
2️⃣ PCF 결과 조회

--------------------------------
[화면 구조]
--------------------------------

데이터 조회 탭 내부에는 **상단 세부 탭 2개**를 만든다.

세부 탭

입력 데이터 조회  
PCF 결과 조회

--------------------------------
1️⃣ 입력 데이터 조회 탭
--------------------------------

목적

프로젝트 내 **모든 원천 입력 데이터**를 확인하는 화면

조회 대상

- 자사 입력 데이터
- 하위차사 제출 데이터
- 적용 배출계수 정보

레이아웃

상단에 **조회 필터 영역**을 배치한다.

필터 구성

- 업체
- 차수
- 납품 항목
- 상태
- 데이터 버전
- 기간

필터 아래에는 **데이터 조회 테이블**을 배치한다.

테이블 컬럼 예

- 업체명
- 차수
- 납품 항목
- 데이터 유형
- 입력 값
- 단위
- 상태
- 마지막 수정일

행 클릭 시  
**데이터 상세 보기 패널**이 우측에 열린다.

--------------------------------
데이터 상세 패널 구성
--------------------------------

섹션1

데이터 정보

- 업체명
- 차수
- 데이터 유형
- 입력 항목

섹션2

입력 값 정보

- 값
- 단위
- 적용 배출계수
- 입력 사용자
- 입력 시간

섹션3

버전 정보

- 데이터 버전
- 이전 버전 비교 버튼

주의

이 화면에서는

- 수정 버튼
- 저장 버튼

을 절대 표시하지 않는다.

--------------------------------
2️⃣ PCF 결과 조회 탭
--------------------------------

목적

PCF 계산 결과를 확인하고  
최종 제출 전 검토하는 화면

레이아웃

상단에 **결과 요약 카드 영역**을 배치한다.

요약 카드

- 총 PCF 값
- 계산 기준 버전
- 마지막 계산 시각
- 하위 데이터 반영률

그 아래에는 **PCF 결과 테이블**을 배치한다.

테이블 컬럼 예

- 업체명
- 차수
- 납품 항목
- 배출량
- 단위
- 기여도(%)
- 계산 버전

행 클릭 시  
우측에 **PCF 결과 상세 패널**이 열린다.

--------------------------------
PCF 결과 상세 패널
--------------------------------

섹션1

PCF 계산 요약

- 총 PCF
- 계산 기준
- 계산 시각

섹션2

배출 기여도 분석

- 활동별 기여도
- 하위차사 기여도

섹션3

계산 로그

- 경고 메시지
- 누락 데이터 여부

--------------------------------
[버전 비교 기능]
--------------------------------

PCF 결과 조회 탭에는  
**버전 비교 기능**을 제공한다.

비교 가능한 항목

- PCF 계산 버전
- 계산 시각
- 총 PCF 변화

비교 방식

두 버전을 선택하면  
변경된 값만 강조 표시한다.

--------------------------------
[UX 규칙]
--------------------------------

데이터 조회 화면은 다음 원칙을 따른다.

1️⃣ 조회 전용 화면이다.

2️⃣ 입력 데이터와 계산 결과는 반드시 분리한다.

3️⃣ 이 화면에서는 어떤 경우에도 데이터 편집 기능을 제공하지 않는다.

4️⃣ 계산 결과는 PCF 산정/전송 탭에서 생성되고  
이 화면에서는 조회만 가능하다.

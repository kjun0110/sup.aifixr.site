AIFIX 협력사 포털 – 데이터 조회 탭 구조 재설계

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 협력사 포털의 데이터 조회 탭은 다음과 같은 구조로 되어 있다.

자사 데이터

하위 협력사 데이터

두 개의 세부 탭으로 나뉘어 있다.

그러나 협력사 사용자는 실제로 자사 데이터와 하위 협력사 데이터를 동시에 확인해야 하므로
이 구조는 UX적으로 비효율적이다.

따라서 데이터 조회 탭을 단일 조회 화면으로 재설계하라.

3️⃣ 조회 결과 구조

조회 결과는 자사 기준 공급망 구조 리스트로 표시한다.

구조 예

우리회사 (현재 사용자)

 ├ 2차 협력사 B
 │    └ 3차 협력사 C

 └ 2차 협력사 D

특징

자사는 항상 최상단

하위 협력사만 표시

동일 차수 다른 회사는 표시하지 않는다.

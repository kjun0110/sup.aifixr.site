너는 제조업 공급망 ESG/PCF 관리 SaaS **AIFIX**의 시니어 프로덕트 디자이너다.

현재 협력사 포털의 **공급망 관리 화면**에는 하위 협력사 구조 조회와 관리 기능만 존재하며,
하위 협력사를 **초대(등록)** 하는 기능이 없다.

1차 / 2차 / 3차 협력사는 모두 자신의 **직접 하위 협력사**를 초대할 수 있어야 한다.

단, 원청은 단일이며 협력사 포털은 **원청 전용 포털**이다.

따라서 아래 규칙에 따라 공급망 관리 화면에 **하위 협력사 초대 기능**을 추가하라.

------------------------------------------------

[1] 공급망 관리 화면 상단 액션 추가

공급망 관리 화면 상단에 Primary 버튼 추가

버튼 이름

"하위 협력사 초대"

버튼 클릭 시 **초대 모달 또는 초대 패널**이 열리도록 한다.

------------------------------------------------

[2] 공급망 트리에서도 초대 가능하도록 추가

공급망 트리 노드에서 다음 액션을 제공

- 협력사 상세 보기
- 보완 요청
- 데이터 보기
- **하위 협력사 초대**

트리에서 특정 협력사를 선택하면
그 협력사의 **하위 차수 협력사 초대**가 가능하도록 한다.

예

1차 협력사
 └ 2차 협력사
      └ + 협력사 초대

------------------------------------------------

[3] 협력사 초대 화면 구성

초대 모달 또는 패널 구성

좌측: 메일 작성  
우측: 발송 기록

--------------------------------

메일 작성 영역

입력 필드

- 협력사 회사명
- 담당자 이름
- 담당자 이메일

입력 방식

- 직접 입력
- CSV 업로드

메일 제목

기본값 자동 생성

"[AIFIX] 공급망 데이터 등록 요청"

메일 본문 기본 템플릿

다음 구조를 기본값으로 자동 생성

--------------------------------

안녕하세요.

귀사는 {현재 사용자 회사명}에 납품하는 협력사로,
해당 납품 제품은 {원청사명} 공급망에 포함되어 있습니다.

공급망 탄소데이터 및 PCF 산정을 위해
AIFIX 시스템 등록이 필요합니다.

아래 링크를 통해 회원가입을 진행해 주시기 바랍니다.

{회원가입 링크}

회원가입 시 DATA CONTRACT 동의가 필요하며,
관련 문서는 아래 첨부파일에서 확인하실 수 있습니다.

감사합니다.

--------------------------------

[4] DATA CONTRACT 자동 첨부

메일 하단에 다음 항목 표시

DATA CONTRACT 버전 선택

예

v2.0 (2026.01)

선택 시 해당 PDF가 **자동 첨부**

버전은 시스템 기준정보에서 관리되며
협력사는 직접 수정할 수 없다.

------------------------------------------------

[5] 발송 기록 영역

우측 패널에 다음 정보를 표시

- 회사명
- 이메일
- 발송 시간
- 상태

상태

- 발송 완료
- 가입 완료
- 발송 실패

가입 완료 시 상태 아이콘을 녹색 체크로 표시

------------------------------------------------

[6] 권한 규칙

1차 / 2차 / 3차 협력사 모두

"직접 하위 협력사"만 초대할 수 있다.

예

1차 → 2차 초대 가능  
2차 → 3차 초대 가능  
3차 → 4차 초대 가능  

하지만 상위 협력사나 동일 차수는 초대할 수 없다.

------------------------------------------------

[7] UX 규칙

초대 기능은 **공급망 관리 기능의 일부**이므로

새로운 네비게이션 메뉴를 만들지 말고

공급망 관리 화면 내부 기능으로 추가한다.

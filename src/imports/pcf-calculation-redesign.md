AIFIX 협력사 포털 – PCF 산정 탭 재설계

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 협력사 포털의 PCF 산정 탭 UI가 원청 시스템과 구조적으로 다르게 설계되어 있다.

그러나 시스템 로직상 모든 협력사(1차 / 2차 / 3차)는 PCF 계산 주체다.

각 협력사는

자신의 데이터 입력 완료

직하위 협력사 데이터 입력 완료 확인

후 PCF 계산을 실행할 수 있다.

따라서 협력사 PCF 산정 화면을 원청 PCF 산정 UI 구조와 동일한 구조로 재설계하라.

단 협력사는 산정 대상 선택 항목이 단순화된다.

1️⃣ 화면 목표

협력사가

PCF 계산 준비 상태를 확인하고

PCF 계산을 실행하고

결과를 확인하고

상위 협력사에게 결과를 전송하는

PCF 계산 콘솔 화면이다.

2️⃣ 상단 – PCF 산정 대상 선택

협력사는 제품 / 공급망 버전 / M-BOM 선택 권한이 없다.

따라서 산정 대상 선택은 산정 기간만 선택하도록 단순화한다.

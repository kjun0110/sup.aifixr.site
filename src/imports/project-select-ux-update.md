너는 제조업 공급망 ESG/PCF 관리 SaaS **AIFIX 협력사 포털**의 시니어 프로덕트 디자이너다.

현재 **프로젝트 선택 화면**에서 작은 UX 개선을 진행하려고 한다.
이미 프로젝트 카드의 데이터 구조(상위차사 규칙 등)는 수정 완료된 상태이며,
이번 작업에서는 **검색 UX 개선과 카드 인터랙션 개선만 수행한다.**

기존 레이아웃과 디자인 스타일은 유지하고, 아래 두 가지 사항만 수정하라.

---

# 1. 검색 placeholder 개선

현재 검색창 placeholder는 다음과 같다.

프로젝트명, 계약번호, 납품 항목으로 검색

하지만 현재 화면의 데이터 구조상 다음 필드도 검색 대상에 포함된다.

* 프로젝트명
* 계약번호
* 납품 항목
* 상위차사

따라서 검색 placeholder를 다음처럼 수정한다.

프로젝트명, 계약번호, 납품 항목, 상위차사 검색

이 수정의 목적은 **검색 가능 데이터와 placeholder 안내 문구를 일치시키는 것**이다.

---

# 2. 검색과 필터 역할 구분 유지

현재 UI에는 다음 필터가 존재한다.

전체 역할
전체 상태

검색과 필터의 역할은 다음처럼 구분된다.

검색

* 프로젝트명
* 계약번호
* 납품 항목
* 상위차사

필터

* 역할 (1차 / 2차 / 3차)
* 상태 (진행중 / 검토중 / 완료)

검색과 필터의 구조는 이미 올바르게 설계되어 있으므로 **UI 위치나 구조는 변경하지 않는다.**

---

# 3. 프로젝트 카드 인터랙션 UX 개선

현재 프로젝트 카드에서 **어디를 클릭해야 프로젝트로 들어가는지 인터랙션이 명확하지 않다.**

다음 UX 규칙을 적용하라.

### 카드 전체 클릭 가능

프로젝트 카드는 **전체 영역이 클릭 가능한 컴포넌트**로 동작해야 한다.

사용자가 카드 어느 부분을 클릭해도 해당 프로젝트 내부 화면으로 이동한다.

---

### Hover 상태 추가

마우스를 카드 위에 올리면 다음 효과를 적용한다.

* 카드 shadow 증가
* 카드 border 또는 배경 강조
* 카드가 약간 떠오르는 느낌의 elevation 변화

예시

기본 shadow
0px 4px 16px rgba(0,0,0,0.05)

hover shadow
0px 8px 24px rgba(0,0,0,0.12)

---

### 클릭 가능성 시각화

Hover 상태에서 다음 중 하나를 추가한다.

* 카드 우측 상단에 작은 **arrow icon**
* 카드 제목 색상 강조
* cursor pointer 적용

목적은 **사용자가 카드가 클릭 가능한 인터랙션 요소임을 직관적으로 인지하도록 만드는 것**이다.

---

# 4. 디자인 시스템 유지

다음 요소는 변경하지 않는다.

* 카드 레이아웃
* 카드 정보 구조
* 색상 팔레트
* 타이포그래피
* AIFIX 디자인 시스템
* 프로젝트 카드 배치 구조
* 네비게이션 구조

이번 작업은 **디자인 리뉴얼이 아니라 UX 디테일 개선 작업**이다.

---

# 5. 작업 범위 제한

이번 작업에서는 다음을 수행하지 않는다.

* 프로젝트 카드 정보 구조 변경
* 새로운 필드 추가
* 네비게이션 구조 변경
* 프로젝트 내부 화면 설계

오직 아래 두 가지만 수정한다.

2️⃣ 프로젝트 카드 인터랙션 UX 개선

AIFIX 협력사 포털 – 데이터 조회/관리 통합 화면 설계

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 협력사 포털의 프로젝트 내부 화면에서
"데이터 조회" 탭과 "데이터 관리" 탭이 분리되어 있다.

원청 포털과 UX 구조를 통일하기 위해
협력사 화면에서도 조회 + 관리 기능을 하나의 탭으로 통합하여 재설계하라.

2. 데이터 관리 화면 목적

협력사 사용자는 이 화면에서 다음 업무를 수행한다.

1️⃣ 자사 데이터 입력 및 수정
2️⃣ 하위 협력사 데이터 확인
3️⃣ 하위 협력사 데이터 상세 조회
4️⃣ 전체 공급망 데이터 Excel 다운로드

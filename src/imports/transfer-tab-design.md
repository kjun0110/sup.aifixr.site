AIFIX – 전송 탭 UI 설계

너는 제조업 공급망 ESG/PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 프로젝트 화면의 네비게이션2에 [전송] 탭이 추가되었다.
이 화면은 사용자가 상위차사에게 **PCF 데이터 패키지(자사 데이터 + 하위 협력사 데이터 + PCF 산정값)**를 전송하기 전에
데이터 준비 상태를 확인하고 전송을 실행하는 화면이다.

이 화면은 단순 이메일 발송 UI가 아니라 데이터 준비 상태 확인 → 전송 실행 워크플로우를 제공해야 한다.

1️⃣ 화면 목적

전송 탭은 다음 역할을 수행한다.

특정 기간(쿼터)에 대한 전송 대상 데이터 상태 확인

자사 데이터 입력 완료 여부 확인

하위 협력사 데이터 검토 완료 여부 확인

PCF 산정 완료 여부 확인

모든 조건이 충족되면 상위차사에게 데이터 전송 실행

2️⃣ 상단 필터 영역

화면 상단에는 전송 대상 기간 선택 필터가 있어야 한다.

필터 항목

기간 (Quarter)
달력 아이콘이 있는 드롭다운

예시

2026 Q1
2026 Q2
2026 Q3
2026 Q4

사용자가 기간을 선택하면
해당 기간 기준으로 전송 준비 상태가 자동 조회된다.

3️⃣ 전송 준비 상태 영역

필터 아래에는 전송 준비 상태 카드를 배치한다.

섹션 제목

전송 준비 상태

이 영역은 3개의 상태 카드로 구성된다.

1️⃣ 자사 데이터 상태

카드 제목

자사 데이터

표시 내용

상태 아이콘 + 상태 문구

상태 예시

🟡 입력 필요
🔵 입력 완료
🟢 검토 완료 / 저장 완료

문구 예시

해당 기간의 자사 데이터 입력 및 검토가 완료되었습니다.
데이터 관리 탭에서 입력된 데이터가 전송 준비 상태입니다.

또는

해당 기간의 자사 데이터가 아직 입력되지 않았습니다.
데이터 관리 탭에서 데이터를 입력하세요.

2️⃣ 하위 협력사 데이터 상태

카드 제목

하위 협력사 데이터

표시 정보

하위 협력사 데이터 검토 상태

예시

🟢 모든 하위 협력사 데이터 검토 완료
🟡 일부 협력사 데이터 검토 필요

세부 표시

협력사 데이터 상태 요약

예시

검토 완료 6개
검토 필요 2개

문구 예시

하위 협력사 데이터 검토 및 수정이 완료되었습니다.
전송 준비 상태입니다.

3️⃣ PCF 산정 상태

카드 제목

PCF 산정

상태 예시

🔴 산정 미실행
🟡 산정 필요
🟢 산정 완료

문구

PCF 산정이 완료되었습니다.

또는

PCF 산정이 아직 실행되지 않았습니다.
PCF 산정 탭에서 계산을 실행하세요.

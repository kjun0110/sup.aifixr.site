AIFIX – 협력사 초대 및 연결 승인 워크플로우 수정

너는 제조업 공급망 ESG/PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 협력사 포털에는 하위 협력사 초대 기능 UI가 이미 존재한다.

현재 구조

공급망 관리 화면 우측 상단에 [하위 협력사 초대] 버튼

버튼 클릭 시 초대 메일 작성 모달 팝업

이메일 발송 후 협력사가 가입하면 즉시 시스템 접근

하지만 공급망 데이터 신뢰성과 관리 구조를 위해
협력사 초대 → 동의 → 가입 → 승인 → 연결 워크플로우로 수정해야 한다.

따라서 현재 UI 구조를 유지하면서 워크플로우와 상태 시스템을 업그레이드하는 방향으로 수정 설계하라.

9️⃣ 알림 시스템

하위 협력사가 가입하면 상위 공급망 회사들에게 알림을 전달한다.

알림 규칙

직상위 협력사 → 승인 요청 알림
상위 협력사 → 가입 알림

예

3차 협력사 가입
↓
2차 → 승인 요청 알림
1차 → 가입 알림
원청 → 가입 알림

동일 차수 협력사

에게는 알림이 전달되지 않는다.

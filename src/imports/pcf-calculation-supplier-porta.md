너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

첨부된 화면은 원청사의 PCF 산정 탭 화면이다.
이 화면의 디자인 구조와 컴포넌트를 기준으로 협력사 포털의 PCF 산정 탭을 재구성하라.

목표는 원청과 협력사가 동일한 PCF 계산 콘솔 UI를 사용하도록 디자인을 통일하는 것이다.

단, 협력사는 원청과 역할이 다르므로 일부 입력 항목만 제거하고 나머지 UI 구조는 동일하게 유지한다.

1️⃣ 디자인 기준

원청 PCF 산정 화면을 Design Reference로 사용한다.

다음 UI 영역은 동일한 레이아웃과 스타일을 재사용한다.
